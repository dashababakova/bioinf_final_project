## Индивидуальная часть

Изначально для выполнения проекта мной был выбран организм **Bombus sonorus (звонкий шмель)**. Однако в процессе подготовки выяснилось, что для данного вида в открытых базах данных отсутствует качественная референсная сборка генома с полноценной аннотацией. В связи с этим было принято решение заменить организм на **Bombus vosnesenskii (желтолицый шмель)**.

## Экология и распространение Bombus vosnesenskii

| Параметр | Значение |
|:---|:---|
| **Ареал** | Запад Северной Америки: от южной Британской Колумбии (Канада) до центральной Калифорнии и Невады (США) |
| **Среда обитания** | Прибрежные леса, чапараль, луга, городские сады, сельскохозяйственные угодья |
| **Высотный диапазон** | От уровня моря до 3000 метров |
| **Температурная толерантность** | Критический минимум (CTmin): зависит от условий сбора; максимальная температура полёта: **~42°C** (перегрев через 3 минуты полёта) |
| **Статус вида** | **LC (Least Concern)** — не находится под угрозой, стабильная популяция |
| **Экологическая роль** | Важный опылитель диких растений и сельскохозяйственных культур (включая томаты, чернику, клюкву) |
